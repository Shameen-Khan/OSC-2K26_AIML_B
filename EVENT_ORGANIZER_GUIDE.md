## ğŸ“ Problem Types

### Medium Level (Syntax & Basic Logical)
1. **Missing colons** after if/for/while/def statements
2. **Incorrect indentation** in code blocks
3. **Missing parentheses** in function calls
4. **Typos** in variable names
5. **Missing quotes** around strings
6. **Type errors** (string + int)
7. **Off-by-one errors** in ranges
8. **Wrong list indexing**
9. **Division by zero**
10. **Missing return statements**
11. **Wrong boolean operators** (and/or)
12. **Mismatched brackets**
13. **Undefined variables**
14. **Wrong comparison operators**

### Advanced Level (Complex Logical)
1. **Mutable default arguments**
2. **Closure variable binding**
3. **Shallow vs deep copy**
4. **Dictionary key errors**
5. **List modification during iteration**
6. **Recursion errors**
7. **Scope issues with global variables**
8. **Class vs instance variables**
9. **Generator exhaustion**
10. **Complex algorithm bugs**
